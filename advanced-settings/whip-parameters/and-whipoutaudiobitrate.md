# \&whipoutaudiobitrate

WHIP Option! ([`&whip`](and-whip.md))

## Aliases

* `&woab`

## Options

Example: `&whipoutaudiobitrate=128`

| Value           | Description           |
| --------------- | --------------------- |
| (integer value) | audio bitrate in kbps |

## Details

`&whipoutaudiobitrate` lets you specify the WHIP audio bitrate (kbps).
